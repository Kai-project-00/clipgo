🔧 콘텐츠 타입 테스트 페이지

ClipGo 확장 프로그램이 다양한 콘텐츠 타입에서 잘 작동하는지 테스트합니다.

💻 코드 블록 테스트

다음은 JavaScript 코드 예제입니다:

// ClipGo 확장 프로그램 초기화 코드
const clipManager = new ClipManager({
    storage: 'chrome.storage',
    categories: ['work', 'personal', 'ideas']
});

// 클립 저장 함수
async function saveClip(text, category) {
    try {
        await clipManager.saveClip({
            text: text,
            category: category,
            timestamp: new Date(),
            source: window.location.href
        });
        console.log('클립이 성공적으로 저장되었습니다.');
    } catch (error) {
        console.error('클립 저장 실패:', error);
    }
}

📝 리스트 형식 테스트

ClipGo 사용 방법

1. 확장 프로그램 설치 - Chrome 웹 스토어에서 ClipGo 확장 프로그램을 설치합니다.
2. 텍스트 선택 - 웹 페이지에서 저장하고 싶은 텍스트를 드래그하여 선택합니다.
3. 카테고리 선택 - 나타나는 팝업에서 적절한 카테고리를 선택합니다.
4. 저장 확인 - 확장 프로그램 팝업에서 저장된 내용을 확인합니다.

🎨 혼합 콘텐츠 테스트

다양한 형식의 콘텐츠가 혼합된 텍스트입니다:

ClipGo는 drag-and-drop 기능을 지원하며, 사용자가 웹 페이지에서 텍스트를 쉽게 저장할 수 있습니다. 이 기능은 특히 연구자와 개발자들에게 유용합니다.

주요 특징:

  • 실시간 동기화
  • 다중 카테고리 지원
  • 내보내기 기능 (JSON, CSV, Markdown)

다음은 간단한 사용 예제입니다:

const clip = {
    text: "저장할 텍스트",
    category: "research",
    timestamp: new Date()
};
