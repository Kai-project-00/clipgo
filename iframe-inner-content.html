Iframe Content

🎯 ClipGo Iframe 테스트

이 텍스트는 iframe 내부에 있습니다. ClipGo 확장 프로그램이 iframe 내부의 콘텐츠에서도 텍스트 선택을 지원하는지 테스트할 수 있습니다.
iframe은 웹 페이지 내에 다른 문서를 포함하는 강력한 기능입니다. 동일 출처 정책에 따라 접근이 제한될 수 있습니다.

📋 테스트 포인트

기능 테스트:
  • 텍스트 드래그 선택
  • ClipGo 팝업 표시
  • 카테고리 선택
  • 저장 기능

💻 코드 예제

// iframe 내부 콘텐츠 감지 function detectIframeContent() { if (window.self !== window.top) { console.log('이 페이지는 iframe 내부에 있습니다'); return true; } return false; } // 텍스트 선택 이벤트 document.addEventListener('mouseup', function() { const selection = window.getSelection(); if (selection.toString().length > 0) { console.log('Iframe 내부 텍스트 선택됨'); } });

🔍 기술적 세부사항

iframe 내부에서의 텍스트 선택은 다음과 같은 기술적 고려사항이 있습니다:
1. 동일 출처 정책: iframe과 부모 페이지가 동일한 도메인이어야 합니다.
2. 콘텐츠 스크립트: 확장 프로그램의 콘텐츠 스크립트가 iframe 내부에도 주입되어야 합니다.
3. 이벤트 전파: iframe 내부의 이벤트가 부모 페이지로 전파되어야 합니다.
